      15. 3Sum

      February 9, 2023 in LeetCode

      算是 Two Sum 的一種另類進階，從 nums 中找出和為 0 的三個 element ，並組成 List of list 。特別注意，不能有兩個內容一樣的 list。因為整個題目並沒有對 nums 的 index 有任何要求，故可以把 nums 排序，為解題拓開另一種思路。

      思路

      對原 nums 進行排序，然後開始 for loop 排序後的 nums 。這裡注意 ! 不是遍歷到最後一個停止，而是到倒數第三個就可以了。 接著處理

      • 三數和小於 0，則將左邊那個指針 j 右移，使得三數和增大一些。
      • 三數和大於 0，則將右邊那個指針 k 左移，使得三數和變小一些。
      • 三數和等於 0，為答案之一，要記錄下來。

      解答

      class Solution {
    public List<List<Integer>> threeSum(int[] nums) {
        Arrays.sort(nums);
        Set<List<Integer>> ans = new HashSet<>();
        for(int i = 0 ; i < nums.length -2 ; i++){
            int j = i + 1;
            int k = nums.length - 1;

            while(j < k){
                if(nums[i] + nums[j] + nums[k] == 0){
                    List<Integer> list = List.of(nums[i] , nums[j] ,nums[k]);
                    ans.add(list);
                    j++;
                    k--;
                } else if( nums[i] + nums[j] + nums[k] > 0){
                    k--;
                } else {
                    j++;
                }
            }
        }
        return ans.stream().map(e -> e).collect(Collectors.toList());
    }
}
